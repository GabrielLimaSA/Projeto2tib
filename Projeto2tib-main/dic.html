Ferramentas pra ajudar na crição de um rpg de mesa

Antes de iniciar essa parte de "Ferramentas" é bom dar a dica inicial pra criação de um Rpg escolher um sistema pra criar sua campanha seja o sistema de D&D, TORMENTA20, ORDEM PARANORMAL RPG,CALL OF CTHULHU, e outras centenas de sistemas pra jogar mais lembre que além desses sistemas é bom não se limitar apenas neles é sempre é uma opção criar seu proprio sistema juntando sistemas já existentes.

Ferramentas que vão te ajudar

Fantasy Name Generator: gerador de nomes para todo tipo de RPG,Acredite tem muita coisa. (//www.fantasynamegenerators.com/)

HeroForge: Criação de Personagens 3D.é melhor pra Rpg's medievais.(https://heroforge.com/)

Tabletop Audio: site que compila várias músicas para RPGs, separadas de acordo com qual sentimento ou apelo você quer evocar em certa cena( https://tabletopaudio.com/)

Newspaper Clipping Generator: criador de "pedaços de notícia de jornal". Bem primitivo, mas fácil de usar(https://www.fodey.com/generators/newspaper/snippet.asp)

Architect Documentos: Crie notícias, panfletos, recados, notas, cartas e pedaços de filmes com esse gerador! (https://handouts.cthulhuarchitect.com/generator)

Text to Handwriting: criador de cartas escritas a mão, para props de RPG. Personalizável, mas ainda está em desenvolvimento (e pode ter alguns bugs)(https://saurabhdaware.github.io/text-to-handwriting/)

Drawio/Diagrams.net: criador de mapas mentais, perfeito para organizar ideias de uma campanha, linhas de plot ou relacionamentos entre NPCs( https://app.diagrams.net/)

RPG Map Editor: criador de mapas de RPG, pode ser usado no navegador ou baixado como software (https://deepnight.net/tools/rpg-map/)

Hextml: criador de mapas no estilo hex, ideal para RPGs "hexcrawl" ou com foco em exploração (https://hextml.playest.net/)

Inkarnate: criador de mapas semelhante à mapas mundi medievias. Pago, mas possui uma versão gratuita para uso - é necessário fazer uma conta no site( https://inkarnate.com/)

Agzaar's Fantasy Map Generator: gerador aleatório de mapas no estilo "mapa mundi". Super completo - cria cidades, rotas, rios e nascentes, e pode ser personalizado de acordo com o interesse do criador.Curiosidade a imagem de fundo das outras abas do site foi feita com ele(https://azgaar.github.io/Fantasy-Map-Generator/)

Here Be Taverns: uma junção de geradores de vários tipos de conteúdo (tavernas, personagens, items, magias, monstros, ganchos de aventura e lugares especiais). Para usar, clique em um deles e os resultados apareçerão (dê F5 para fazer o gerador criar novos resultados)(https://www.herebetaverns.com/#browse-free)

Cthulhu Architect Mapas: Baixe gratuitamente mapas para cenários de rpgs modernos.(https://www.patreon.com/cthulhu_architect/posts?filters%5Btag%5D=free%20maps)

Rodeo 2.0: Ainda ta em Beta mas é a versão atualizada do tabletop Gratuito Owlbear. Basicamente uma mesa virtual. (https://www.owlbear.app/)
